# Fish shell

## Install

1. Install [Fish](https://fishshell.com/) for your OS.
2. [Optional] If using NeoVim, update the `shell` key to the executable of Fish.
eg. `shell /bin/fish`.
3. Copy `config.fish` and all the `functions/` over into their respective
   directories in `~/.config/fish/`.
4. [Optional] If using NeoVim, you may need to add the executable to the PATH in
   the Fish config.
eg. `fish_add_path /opt/nvim-linux-x86_64/bin/`.
